# Flexbox

* Nos permite posicionar os elementos dentro da caixa.
* Controle em uma dimensao (horizontal e vertical)
* Alinhamento, direcionamento, ordenar e tamanhos

```css
div.parent {
    diplay: flex;
}
```

## flex-direction

* Qual a direcao do flex: horizontal ou vertical
* row | column

## alinhamento

* justify-content
* align-items
