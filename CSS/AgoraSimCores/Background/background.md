## Background

- Define um fundo para o nosso elemento
- Sua area de atuacao e a caixa toda
- por padrao, e transparente

### Exemplos

- Usar cores solidas
- Usar imagens
- Controlar
    - a posicao das imagens
    - se elas se repetem ou nao
    - o tamanho delas na caixa
- Usar cor e imagem juntas
- Usar cor gradiente
